📋 ОПИСАНИЕ
--------------------------------------------------------------------------------
Автоматический загрузчик видео с HDRezka с поддержкой многопоточности, 
HLS-потоков, автоматического поиска озвучек и анализа доступных сезонов/серий.

✨ ВОЗМОЖНОСТИ
--------------------------------------------------------------------------------
🎥 Многопоточная загрузка          — До 20 потоков для максимальной скорости
📺 HLS поддержка                   — Автоматическое скачивание .m3u8 потоков
🎙 3 метода поиска озвучек         — API → HTML → Сканирование ID (до 3000+)
📊 Анализ сезонов/серий            — Показывает доступные сезоны для каждой озвучки
🔤 Динамический поиск              — Параметр --sca-id для настройки диапазона
📁 Умные имена файлов              — Название [1080p]/S1/Название [1080p]_s01e01.mp4
🎨 Прогресс-бар Ubuntu-style       — С зелёными | и перезаписью строки
🔧 Склонение слов                  — Правильное: "1 сезон", "2 сезона", "5 сезонов"
🌐 Кроссплатформенность            — Windows, Linux, macOS, Android (Termux)
⌨️ Клавиша 'S'                     — Остановка после текущей серии
🔐 Авторизация                     — Поддержка аккаунта HDRezka

📦 УСТАНОВКА
--------------------------------------------------------------------------------
1. Требования:
   - Python 3.7+
   - ffmpeg (опционально, для HLS)

2. Установка зависимостей:
   pip install HdRezkaApi requests beautifulsoup4

3. Установка ffmpeg (опционально):
   
   Windows:
   - Скачать с https://ffmpeg.org/download.html
   - Добавить в PATH
   
   Linux:
   sudo apt install ffmpeg
   
   macOS:
   brew install ffmpeg

🚀 ИСПОЛЬЗОВАНИЕ
--------------------------------------------------------------------------------
🔹 Этап 1: Получить список озвучек
   python hdrezka-dl_v4.6.5.py "URL" -u email -p pass -s 1 -e 1 --get-translations

🔹 Этап 2: Скачать с выбранной озвучкой
   python hdrezka-dl_v4.6.5.py "URL" -u email -p pass -q 1080p -s 1 --all -t 56 --threads 20

🔹 Скачать несколько сезонов
   # Сезоны 1-3
   python hdrezka-dl_v4.6.5.py "URL" -u email -p pass -q 1080p -s "1-3" --all -t 56 --threads 20
   
   # Сезоны 1, 3, 5
   python hdrezka-dl_v4.6.5.py "URL" -u email -p pass -q 1080p -s "1,3,5" --all -t 56 --threads 20

🔹 Расширенный поиск озвучек
   # По умолчанию (до ID 1000)
   python hdrezka-dl_v4.6.5.py "URL" -u email -p pass --get-translations
   
   # Расширенный поиск (до ID 3000)
   python hdrezka-dl_v4.6.5.py "URL" -u email -p pass --get-translations --sca-id 3000
   
   # Быстрый поиск (до ID 500)
   python hdrezka-dl_v4.6.5.py "URL" -u email -p pass --get-translations --sca-id 500

📋 АРГУМЕНТЫ КОМАНДНОЙ СТРОКИ
--------------------------------------------------------------------------------
| Аргумент              | Описание                                      | По умолчанию     |
|-----------------------|----------------------------------------------|------------------|
| url                   | 🔗 Ссылка на страницу HDRezka                | Обязательный     |
| -u, --username        | 👤 Email/логин                               | Обязательный     |
| -p, --password        | 🔑 Пароль                                    | Обязательный     |
| -q, --quality         | 🎥 Качество (360p/480p/720p/1080p/1080p_ultra)| 1080p            |
| -s, --season          | 📺 Сезон: '1', '1-3', '1,3,5'                | 1                |
| -e, --episode         | 🎬 Серия: '1', '1-10', '1,3,5'               | Все              |
| -t, --translation     | 🎙 Озвучка: ID или название                  | По умолчанию     |
| --get-translations    | 📋 Получить список озвучек                   | —                |
| --save-translations   | 💾 Сохранить озвучки в JSON                  | —                |
| --all                 | 📦 Скачать весь сезон                        | —                |
| --threads             | 🧵 Количество потоков (1-20)                 | 8                |
| --hls-threads         | 🧵 Потоков для HLS                           | 4                |
| --delay               | ⏱ Задержка между сериями (сек)              | 2                |
| -o, --output          | 📁 Папка для сохранения                      | ~/Downloads/HDRezka |
| --dry-run             | 🔍 Тест без скачивания                       | —                |
| --force-range         | ⚡ Принудительный Range                      | —                |
| --no-progress         | 🚫 Отключить прогресс-бар                    | —                |
| --max-seasons         | Макс. сезонов для анализа                    | 20               |
| --max-episodes        | Макс. серий в сезоне                         | 100              |
| --sca-id              | 🔍 Диапазон сканирования озвучек             | 1000             |
